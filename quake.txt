** FLOW **
|-- init
|   |- init glfw
|   |- init opengl
|   |- set window icon
|   |- load shaders
|   |- load textures
|   |- init camera
|   |- init scene
|   |  |- init clock
|   |  |- init planets
|   |  |  |- load objects (.orb files)
|   |  |  |- init planets
|   |  |- calc planets init positions (current date and time)
|   |  |- init ui
|   |  |  |- init orbits
|   |  |  |- init rings
|   |  |  |- init labels
|   |  |- load skybox
|   |- init sound
|-- update
|   |- record fps
|   |- handle inputs
|   |- update scene
|   |  |- update global simulation clock
|   |  |- update position, rotation and scale for each planet
|   |  |- update position for each label (text)
|   |- render scene
|   |  |- draw planets
|   |  |- draw ui
|   |  |  |- draw orbits
|   |  |  |- draw rings
|   |  |  |- draw labels
|   |  |- draw skybox
|-- stop
|   |- free dynamic arrays
|   |- terminate glfw
